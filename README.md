# eslint-config

![badge](https://github.com/fiahfy/eslint-config/workflows/Node.js%20Package/badge.svg)

> My ESLint configs including typescript and prettier.

## Installation

```bash
npm install @fiahfy/eslint-config -D
```

## Usage

```bash
npm install @fiahfy/eslint-config -D
```

Then, in your `.eslintrc.js`

```js
module.exports = {
  extends: ["@fiahfy"],
}
```
